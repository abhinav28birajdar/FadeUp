## Features

### For Customers
- 📍 **Discover Nearby Shops** - Find barber shops near you with GPS-powered search
- ⏱️ **Live Queue Tracking** - Join queues remotely and track your position in real-time
- 📅 **Smart Booking** - Schedule appointments with your favorite barbers
- 🔔 **Push Notifications** - Get notified when it's almost your turn
- ⭐ **Reviews & Ratings** - Read and write authentic reviews
- ❤️ **Favorites** - Save your favorite shops for quick access
- 🌙 **Dark Mode** - Beautiful UI that adapts to system theme

### For Barbers
- 📊 **Live Dashboard** - Manage queues and bookings in one place
- 👥 **Customer Management** - Track customer history and preferences
- 💈 **Service Management** - Create services with pricing and duration
- 📈 **Analytics** - Monitor earnings and business performance
- 🎨 **Shop Customization** - Upload photos and update shop details
- 💬 **Customer Engagement** - Respond to reviews and build relationships

## About

FadeUp is a production-ready React Native application built with Expo SDK 54, TypeScript, and Supabase. It features real-time queue updates, secure authentication, optimized performance with FlashList, smooth 60fps animations using Reanimated, and a modern UI with light/dark mode support. The app includes comprehensive Row-Level Security (RLS) policies, secure credential storage with Expo SecureStore, and offline-capable data caching for a seamless user experience.
