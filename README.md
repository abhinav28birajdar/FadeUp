## ✨ Features

**For Customers:**
- 🗺️ Find nearby barber shops
- � Book appointments easily
- ⏰ Real-time queue tracking
- ⭐ Reviews and ratings

**For Shop Owners:**
- 📊 Live dashboard
- 👥 Queue management
- 📝 Booking management
- 💬 Customer feedback

## 🛠️ Tech Stack

- **Frontend:** React Native with Expo
- **Backend:** Supabase
- **Styling:** NativeWind (TailwindCSS)
- **State:** Zustand
- **Maps:** React Native Maps
