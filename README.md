**FadeUp** is a high-performance, production-ready platform serving Customers, Barbers, and Admins. Built from the ground up to provide a seamless, real-time experience in the urban grooming space.

## 🛠️ Advanced Tech Stack
- **Framework:** [Expo SDK 54](https://expo.dev/) (Managed Workflow)
- **Engine:** React Native 0.81 (Hermes)
- **Backend:** [Firebase 11+](https://firebase.google.com/) (Firestore, Auth, Storage)
- **State:** Zustand 5.0 (Atomic state updates)
- **UI:** Custom design system with Lucide icons
- **Environment:** Expo Router v6 (File-based navigation)

## 📋 Key Modules & Features

### 👤 Customer Experience
- **Smart Discovery:** Discover nearby shops with real-time ratings and addresses.
- **Live Queue:** Track your position and estimated wait time in real-time.
- **Wallet Integrated:** Seamless credit management and referral rewards.
- **Secure Messaging:** Chat directly with your barber for consultations.

### ✂️ Barber Manager (Professional)
- **Real-Time Queue:** One-click customer management (Arrived, In Service, Done).
- **Earnings Tracker:** Real-time visibility into daily and monthly performance.
- **Shop Analytics:** Dynamic services and pricing management.
- **Availability Controls:** Instant shop-open/close toggles.

### 🛡️ Admin Suite
- **Merchant Management:** Approve/Review newly registered shops.
- **Platform Integrity:** Global user management and moderation tools.
- **System Monitors:** Real-time visibility into platform activity.

## 🏗️ Project Initialization
To start the project or refresh the environment:

1.  **Clone & Install**: `npm install`
2.  **Environment**: `cp .env.example .env` (Populate with Firebase keys)
3.  **Local Dev**: `npx expo start`
4.  **Native Update**: `npx expo prebuild`
